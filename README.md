# Ishan Rawat | Persona 3 Portfolio

A Persona 3-themed interactive portfolio website built with React + Vite.

## Live Demo

[https://ishanrawat97.github.io/persona-resume/](https://ishanrawat97.github.io/persona-resume/)

## Pages

- **About Me** - Introduction, core skills, and domain expertise with reveal panels
- **Resume** - Education, technical skills, professional experience, and achievements
- **Socials** - LinkedIn, GitHub, and contact info
- **Side Projects** - Airline Tracker, Stockfish Move Evaluator, Quick Commerce Price Comparator
- **GitHub** - Direct link to GitHub profile

## Controls

| Key | Action |
|-----|--------|
| `Arrow Up / Down` | Navigate menu items |
| `Enter` | Confirm / Reveal |
| `Arrow Left / Right` | Navigate panels |
| `Escape` | Go back |

## Tech Stack

- React 18
- Vite
- React Router (HashRouter)
- Framer Motion
- CSS animations + clip-path styling

## Setup

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run build
npx gh-pages -d dist
```
